## 🚀 部署就绪状态

### 编译要求
- [x] Visual Studio 2022 (MSVC v143)
- [x] CMake 3.20+
- [x] Windows SDK 10.0.22621+
- [x] 可选: SQLite3, llama.cpp

### 构建流程
```bash
mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -A x64 -DQI_ENABLE_AI=ON -DQI_BUILD_TESTS=ON
cmake --build . --config Release
cmake --install . --config Release  # 需要管理员权限
```
