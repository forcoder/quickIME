# 更新日志

## [v1.0.0] - 2026-05-20

### 新增功能
- **模块5: 离线AI推理接口** ✨
  - 实现InferenceEngine核心类，支持AI建议生成
  - 支持多种模型格式（llama.cpp .gguf, ONNX Runtime .onnx）
  - 提供四种AI类型：短句回复、工作话术、日常用语、专业文案
  - 固定提示词模板确保输出质量（2-15字限制）
  - 置信度评估和模型管理功能
  - 完整的错误处理和日志记录

### 文件结构
```
include/quickinput/ai/
├── inference_engine.h      # AI推理引擎头文件
├── prompt_templates.h      # 提示词模板管理器头文件
└── model_loader.h          # 模型加载器头文件

src/ai/
├── inference_engine.cpp    # AI推理引擎实现
├── prompt_templates.cpp    # 提示词模板管理器实现
├── model_loader.cpp        # 模型加载器实现
└── test_ai.cpp            # 测试程序

docs/
└── Module5_AI_Inference.md # 详细文档
```

### CMake配置更新
- 添加QI_ENABLE_AI选项控制AI模块编译
- 自动检测llama-cpp库（可选依赖）
- 创建独立的quickinput_ai静态库
- 添加test_ai可执行文件用于测试

### API设计特点
1. **线程安全**: 所有公共接口都是线程安全的
2. **输入验证**: 所有方法都有严格的输入验证
3. **固定模板**: 使用预定义的提示词模板确保输出一致性
4. **置信度评估**: 为每个AI建议提供0-1.0的置信度评分
5. **资源管理**: 完善的初始化/清理机制
